# Snake

## Версия Unity

- `2022.3.62f2`

## Основные правила игры

- Игрок управляет Змейкой.
- Змейка движется непрерывно в одном направлении.
- Игра разделена на уровни.
- Средняя продолжительноть каждого уровня — 60 секунд. Может варьироваться в завимости от количества яблок на уровне: от 30 секунд до 90 секунд.
- Экран уровня ограничен по четырём краям непроходимой стеной, при столкновении с которой происходит проигрыш.
- На старте уровня змейка состоит из трёх сегментов. 
- На старте каждого уровня змейка стартует с нижней границы экрана по центру и движется вверх по экрану.
- На игровом поле заранее расставлена еда — яблоки. На первом уровне их шесть, далее количество яблок от уровня к уровню возрастает.
- Цель: собрать все яблоки за отведённое на уровне время.
- Когда змейка съедает 1 яблоко, она увеличивается в длину на один сегмент, её движение немного ускоряется.
- Резко развернуться на 180° змейка не может (например, сразу из «влево» в «вправо»).
- Победа на уровне: собраны расставленные на уровне яблоки.
- Поражение: змейка врезается в стену или сталкивается сама с собой.
- После прохождения одного уровня запускается следующий, с другим количеством и расположением целей.

## Управление

- Движение: `Arrow Keys` или `WASD`.
- В мобильной версии движение осуществляется тапом по кнопкам направления в нижней части экрана.
- При победе на уровне в центре экрана появляется кнопка с надписью "Next Level", при тапе по которой запускается следующий уровень.
- При поражении на уровне в центре экрана появляется кнопка с надписью "Try Again", при тапе по которой уровень перезапускается заново.
